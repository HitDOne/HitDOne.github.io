Ensamblador cruzado

Un ensamblador cruzado es un tipo de ensamblador que se utiliza para crear programas en lenguaje de ensamblador para una arquitectura de procesador diferente a la de la computadora donde se ejecuta el ensamblador. Esto permite desarrollar software para sistemas que no tienen las capacidades o los recursos para soportar un entorno de desarrollo propio, como consolas de juegos, dispositivos electrónicos o sistemas embebidos.

El funcionamiento de un ensamblador cruzado es similar al de un ensamblador normal, pero con la diferencia de que el código de máquina generado es compatible con el procesador destino, no con el procesador origen. Para ello, el ensamblador cruzado debe conocer las características y el conjunto de instrucciones del procesador destino, así como el formato de los archivos ejecutables o de los medios de almacenamiento que se utilizarán
