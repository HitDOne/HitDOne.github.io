El assembler es un lenguaje de programación de bajo nivel que se usa para escribir instrucciones precisas para el procesador de un ordenador. Un programa llamado ensamblador se encarga de traducir el código escrito en assembler al código de máquina binario que puede ejecutar el procesador. El assembler es muy útil para programar tareas críticas del sistema operativo, controladores de dispositivos, o dispositivos programables. Sin embargo, el assembler también tiene algunas desventajas, como su dificultad de comprensión, su falta de portabilidad, y su dependencia de la arquitectura del procesador.
